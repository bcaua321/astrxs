## ✅ Run Project
- The first thing is to copy the [.env.example](https://github.com/bcaua321/astrxs/blob/main/.env.example), after that, rename the filename to ".env"
  
- Now, you have to generate an API KEY in [Nasa APIS](https://api.nasa.gov#signUp), and substitute the value in API_TOKEN

- Finally execute the commands:
  
```bash
# Install the dependencies 
$ npm install

# Then, run expo
$ npx expo start
```

Just choose to run in a android device and enjoy. 😊
